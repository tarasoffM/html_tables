HTML Table Reference

Table Layout

<thead> Table Header
<tbody> Table Body
<tfoot> Table Footer
The table is made up of 3 sections. The Header, Body, and Footer. Not all tables require headers and footers howerver it's a good idea to use them for formatting purposes.

Table Tags

Tag Name Description
<table> Table Defines a table
<tr> Table Row Defines a row in a table
<th> Table Header Defines a header cell in a table
<td> Table Data Defines a cell in a table
<thead> Table Header Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Footer Groups the footer content in a table
These are the basic tags used to create a table. The table tag is the parent tag and the tr, th, and td tags are the children tags.

Table Attributes

Attribute Description
border Specifies the width of the border
cellpadding Specifies the space between the cell wall and the cell content
cellspacing Specifies the space between cells
width Specifies the width of a table
height Specifies the height of a table
align Specifies the alignment of a table
bgcolor Specifies the background color of a table
bordercolor Specifies the color of the border
bordercolordark Specifies the color of the dark border
bordercolorlight Specifies the color of the light border
colspan Specifies the number of columns a cell should span
rowspan Specifies the number of rows a cell should span
These are the basic attributes used to style a table. The border attribute is the most commonly used attribute.
